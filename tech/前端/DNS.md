# DNS的定义：
 DNS是域名系统的缩写。

# DNS的作用：
 从域名中解析出IP地址

# DNS的组成部分:
 1.名称解析器
 2.域名空间
 3.名称服务器:名称服务器有3种主要类型
  (1)主名称服务器
  (2)次名称服务器
  (3)惟高速缓存名称服务器

# 域名解析的方式：
 有两种，正向解析和反向解析。
 (1)正向解析：根据域名查询对其对应的IP地址或其他相关信息。
 (2)反向解析：根据主机的IP地址查询其对应的域名。
