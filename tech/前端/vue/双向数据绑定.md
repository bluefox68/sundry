# 数据双向绑定：数据劫持+订阅发布。
  (一)基础概念：
    1.监听器(数据):
    2.编译器(指令):

    3.订阅者:
    4.订阅器:
      管理订阅者
      添加订阅者到订阅器
      通知订阅者更新自身
    5.发布者:

  (二)原理:
    分为两个阶段：
    1.vue初始化的时候，调用监听者的初始化方法，劫持数据对象。在访问数据对象的时候，将订阅者添加到订阅器。在数据对象更新的时候，调用订阅器的通知方法，通知所有订阅者调用它们自身的数据更新方法进行数据更新。
    2.扫描节点下的指令，然后进行匹配，匹配到后生成订阅器(需要传this对象，订阅的属性，回调函数)。
