# 病因：外界刺激导致。
  具体来说，由于结膜组织上的血管和淋巴组织丰富，而结膜组织大部分又跟外界接触，所以外界的抗原很容易使其致敏，导致炎症。

# 症状：
  1.感觉上：
    眼部干涩、 痒，眼睑重坠难睁，眼疲劳
  2.物理上
   (1)白天眦部可见白色泡沫状分泌物
   (2)睑结膜轻度充血、粗糙，有滤泡，血管纹理不清，严重时球结膜也有充血

# 影响：
  结膜炎本身对视力影响一般并不严重，但是当其炎症波及角膜或引起并发症时，可导致视力的损害。

# 相似病症:
  干眼症
